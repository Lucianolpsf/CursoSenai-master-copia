## Orientações Gerais

Faça a criação de um CRUD de produtos de uma confeitaria

## Funcionalidades

* Crie uma classe para o fornecedor dos produtos
* Crie uma classe com os atributos básicos de todos os produtos: como açúcar, trigo....
* Crie a classe produto que herde da classe das matérias primas
* Os produtos precisaram estar vinculados a um fornecedor, cada produto com um fornecedor
* Defina os atributos como privados
* Crie os métodos get e set correspondentes (podendo ser utilizados métodos normais ou decoradores)
* Implemente os métodos CRUD para produtos: Criação, Leitura, Alteração e Exclusão
* Importe o pacote da classe no arquivo de execução e demonstre o uso das funcionalidades
